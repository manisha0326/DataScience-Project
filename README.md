# Human Mood Prediction
Human Mood Prediction is a machine learning project that analyzes text input and predicts the emotional mood expressed in the sentence. The system uses Natural Language Processing (NLP) techniques to process textual data and classify it into different mood categories such as happiness, sadness, anger, fear, surprise, and others.
